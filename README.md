# Cocacola-Dashboard
TITLE: Cocacola Dashboard

Description:
Coca-Cola Sales Analysis Dashboard (Power BI)
Developed an interactive sales dashboard using Microsoft Power BI to analyze Coca-Cola sales data and identify key business insights. The dashboard incorporates advanced visual features such as Key Influencers to determine factors impacting sales performance, Q&A for natural-language data queries, Matrix visuals for detailed comparative analysis, and Map visualizations to display regional sales distribution. This project enables users to explore trends, monitor performance metrics, and make data-driven decisions through an intuitive and dynamic interface.
